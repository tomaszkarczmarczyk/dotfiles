# Dotfiles

This repository contains my personal configuration files for various tools and applications.

The layout is clean and predictable, with a modular structure that makes it simple to understand, extend, and adjust. My goal is a setup that feels practical for everyday use while remaining easy to **replicate** on any machine.

These dotfiles are structured for **GNU Stow**, allowing for easy symlinking into the correct locations without cluttering the system.

## Philosophy

This setup focuses on simplicity, clarity, and long-term maintainability. Instead of treating configuration as a one-off task, this repository keeps everything central so changes remain organized and intentional.

## Highlights

- Modular and easy-to-navigate layout
- Portable structure for reproducible setups
- Clean organization for long-term maintenance
- Effortless deployment with GNU Stow
